# StrGraph
The MLP Interview Project


## Dependencies
- `pybind11`
    - to install:
    ```
    pip3 install pybind11
    ```
    or
    ```
    conda install -c conda-forge pybind11
    ```
- `cmake`
    - to install:
    ```
    brew install cmake
    ```
    or
    ```
    sudo apt install cmake
    ```
